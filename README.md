# Terraform AWS 3-Tier High Availability Infrastructure

This repository contains Terraform code to provision a **3-tier web application infrastructure** on **AWS**, designed for **high availability** and **scalability** across **two Availability Zones**.

The architecture includes:

- **Public Tier**
- **Private Tier**
- **Data Tier**
- **Networking**

This project focuses on demonstrating **Infrastructure as Code (IaC)** best practices, **modular design**, and **multi-AZ resilience** using Terraform.

> **Note:** This is a portfolio project for showcasing Terraform and AWS IaC concepts. No actual application is deployed.
